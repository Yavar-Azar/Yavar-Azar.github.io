# first steps in modeling

## Python (Best for scripting, data analysis, and visualization)
Python and its data analysis and visualization libraries (scipy, numpy, matplotlib, pandas, scikit-learn, etc.) is the best choice for post-processing and representation of your research. Python is free, has a  big community and you can find a large number of projects related to your work there.
## Basics of Density functional theory
As the main workhorse for computational material science, DFT yields a relatively exact description from different properties of the system at quantum-level.
